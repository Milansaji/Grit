## Overview

**GRIT** is an **opinionated backend framework for Go** designed to help developers build  
**clean, scalable, production-ready REST APIs** with minimal effort.

It eliminates repetitive backend patterns by providing:
- automatic CRUD operations
- built-in authentication
- sensible conventions
- a clean, extensible architecture

### Automatic CRUD
- Create, read, update, delete generated automatically
- Pagination and partial updates
- Soft delete by default
- Get-by-ID included
